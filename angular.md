### [Angular](http://angular.io) commands


#### Create a project
````bash
ng new name-project
````

#### Open a Angular project
````bash
cd name-project
ng serve --open
````

#### Create a component
````bash
ng generate component name-component
````

#### Create a service
````bash
ng generate service name-service
````

#### Create a module
````bash
ng generate module name-module
````
